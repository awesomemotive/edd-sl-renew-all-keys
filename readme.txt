Provides an option for customers to easily renew all license keys. Requires Software Licensing for Easy Digital Downloads

== Description ==

This is a simple add-on for [Easy Digital Downloads - Software Licensing](https://easydigitaldownloads.com/downloads/software-licensing/). It does not function without it.

Once activated, a "Renew" button with a select drop down will be added to the top of the [edd_license_keys] shortcode. This will allow customers to easily renew all of their license keys by adding all keys to the cart with a single click. Customers can renew all expired keys, all keys expiring within 30 days, or all keys, regardless of the expiration date.

== Installation ==

1. Purchase and install [Easy Digital Downloads - Software Licensing](https://easydigitaldownloads.com/downloads/software-licensing/). 3.4 or later is required.
2. Add [edd_license_keys] to a page.
3. Renew all option will be shown at the top of the page with [edd_license_keys]
